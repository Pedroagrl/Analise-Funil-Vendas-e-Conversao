# Decisões Analíticas do Projeto

Este projeto foi desenvolvido com o objetivo de analisar o desempenho do funil de vendas e conversão, utilizando SQL e PostgreSQL para estruturação dos dados e Excel para construção do dashboard final.

Durante o desenvolvimento, foram aplicadas boas práticas de organização dos dados, definição de indicadores e construção de análises, com foco em simular um cenário real de análise comercial.

---

## Estruturação da Base Analítica

A base de dados foi construída utilizando SQL, consolidando informações de diferentes tabelas.

Foram integrados dados de:

- Vendas  
- Leads  
- Produtos  
- Lojas  
- Clientes  
- Visitas ao site  

As tabelas foram organizadas de forma a permitir análises cruzadas e geração de indicadores consistentes.

---

## Definição dos Indicadores

Os indicadores foram definidos com foco em análise de performance comercial e eficiência do funil de vendas.

Principais métricas:

- Receita total  
- Quantidade de leads  
- Quantidade de vendas  
- Taxa de conversão (%)  
- Ticket médio  
- Volume de visitas  

O objetivo foi permitir avaliar:

- Eficiência do funil  
- Desempenho comercial  
- Comportamento dos usuários  

---

## Uso de Agregações e Agrupamentos

As análises foram construídas utilizando funções SQL como:

- SUM (receita)
- COUNT (leads e vendas)
- AVG (ticket médio)
- GROUP BY (segmentação por tempo, região e categoria)

Essas operações permitiram consolidar os dados em diferentes níveis analíticos.

---

## Organização das Análises

Os gráficos e tabelas foram estruturados para responder perguntas como:

- Como a receita evolui ao longo do tempo?  
- Qual a taxa de conversão do funil?  
- Quais estados possuem maior volume de vendas?  
- Quais marcas e lojas performam melhor?  
- Como o comportamento de visitas impacta as vendas?  

---

## Organização Visual

O dashboard foi construído no Excel com foco em clareza e leitura executiva.

Foram aplicadas boas práticas como:

- Separação de indicadores principais  
- Uso de gráficos para evolução temporal  
- Rankings para análise comparativa  
- Distribuição das informações por bloco analítico  

O objetivo foi facilitar a interpretação dos dados e apoiar a tomada de decisão.
